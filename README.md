# Visa Global (rebuild)

## Run locally
1) npm install
2) npm start

Open:
- Public: http://localhost:3000/
- Check:  http://localhost:3000/check
- Admin:  http://localhost:3000/admin/login

## Railway variables (recommended)
NODE_ENV=production
JWT_SECRET=<long random>
ADMIN_USER=<your admin>
ADMIN_PASS=<your password>
APP_NAME=Visa Global
